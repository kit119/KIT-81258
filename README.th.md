# KIT-81258  
## พระไตรปิฎกเกาหลี (Tripitaka Koreana)  
### ปริวรรตดิจิทัลเพื่อการอ่านและการสวด (โครงการทดลอง)

ไฟล์ชุดนี้คือ  
**โครงการส่วนตัวของเดฟไทยคนหนึ่ง**  
ไม่ใช่งานสถาบัน  
ไม่ใช่งานทางการ  
ไม่ได้รับการรับรอง  
และไม่อ้างว่าถูกต้องสมบูรณ์

สิ่งเดียวที่อ้างได้คือ  
**ตั้งใจให้คัมภีร์ยังมีเสียงอ่าน**

---

## ที่มาของโครงการ

ผมเคยไปวัดแฮอินซา  
ไม่ได้ตั้งใจจะไปด้วยซ้ำ  
ว่างงานประชุม  
ขึ้นรถเมล์ผิดสาย  
อารมณ์นักท่องเที่ยวหลงทาง

รถพาขึ้นเขา  
เข้าวัด  
เข้าถ้ำ  
ไม่รู้ว่าอยู่ตรงไหน  
ไม่รู้ด้วยซ้ำว่าใกล้พระไตรปิฎกไม้ระดับโลกแค่ไหน

ไม่ได้เข้าไปดูพระไตรปิฎก  
ไม่ได้รู้อะไรเลย

รู้แค่ว่า  
พระคุณเจ้าที่นั่น  
เลี้ยงข้าว  
เลี้ยงน้ำ  
พูดดี  
เมตตา  
และช่วยคนหลงทางคนหนึ่งโดยไม่ถามอะไร

ผมจำวันนั้นได้ดี

---

## สิ่งที่ผมมารู้ทีหลัง

พระไตรปิฎกเกาหลีคือ:
- มรดกโลก
- ความภาคภูมิใจของมนุษยชาติ
- คัมภีร์ที่สมบูรณ์ที่สุดชุดหนึ่งในโลก

แต่ในความเป็นจริงวันนี้:
- คนเกาหลีส่วนใหญ่อ่านฮันจาไม่ออก
- สวดไม่ได้
- อ่านไม่ได้
- คัมภีร์มีอยู่ แต่เงียบ

มันยังอยู่  
แต่เหมือน **ไม่มีปาก**
