## Source Data

The file `koreana.zip` used in this project is derived from publicly available
digital resources related to the Korean Tripitaka (Daejanggyeong).

Primary reference:
- Digital Dictionary of Buddhism (DDB): http://www.buddhism-dict.net/ddb/
- Download source: http://www.acmuller.net/download/daejanggyeong.zip

This project does not claim ownership of the source texts.
The data is used solely for non-commercial, research, and cultural
transliteration experiments.
